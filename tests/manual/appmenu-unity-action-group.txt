Test Unity Action Group
=======================

These tests show that the Unity action group is working
GTK+ 3 without the menu proxy patch must be installed
unity-gtk-module must be installed

Test Gnome Terminal Menus
-------------------------

#. Start Gnome Terminal
#. Select Help

Outcome
 Contents and About must be sensitive (not greyed out, enabled)

Test Devhelp Menus
------------------

#. Start Devhelp
#. Select Window

Outcome
 New Tab, Print, and Close must be sensitive (not greyed out, enabled)

Test Eye of GNOME
-----------------

#. Start Eye of GNOME
#. Select Help

Outcome
 Contents and About must be sensitive (not greyed out, enabled)
